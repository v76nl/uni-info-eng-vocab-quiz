# uni-info-eng-vocab-quiz

講義「情報英語」の語彙テスト対策用Webアプリ。  
グループでの単語リスト作成のため、Googleスプレッドシートを使用している。  
<https://v76nl.github.io/uni-info-eng-vocab-quiz/>

## 使い方

1. Google スプレッドシートに単語と意味を入力
2. 数秒で自動的にクイズが更新

## 開発環境の実行

パッケージのインストール：

```bash
npm install
```

開発サーバーの起動：

```bash
npm dev
```

ビルド：

```bash
npm build
```
