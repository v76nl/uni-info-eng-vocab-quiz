# uni-info-eng-vocab-quiz

講義「情報英語」の語彙テスト対策用Webアプリ。  
グループでの単語リスト作成のため、Googleスプレッドシートを使用している。  
<https://v76nl.github.io/uni-info-eng-vocab-quiz/>

## 使い方

1. Google スプレッドシートに単語と意味を入力
2. 数秒で自動的にクイズが更新

## 開発環境の実行

パッケージのインストール：

```bash
pnpm install
```

開発サーバーの起動：

```bash
pnpm dev
```

### スマホ実機でのプレビュー方法

#### 方法A：同一Wi-Fiからアクセスする (一般的なネットワーク用)

PCとスマホが同じWi-Fiルーター等に接続されている場合は、ネットワークに公開してアクセスできます。

1. 以下のコマンドでサーバーを起動する：
    ```bash
    pnpm dev --host
    ```
2. ターミナルに表示される `Network: http://<PCのIPアドレス>:5173/` のURLにスマホのブラウザからアクセスする。

#### 方法B：トンネリングツールを使う (学内Wi-Fiなど端末間通信が制限されている場合)

学内ネットワークなど、同一Wi-Fi内でも端末同士の直接アクセスが禁止されている場合は、一時的な公開トンネルを利用する。

1. 通常通りサーバーを起動する：
    ```bash
    pnpm dev
    ```
2. 別のターミナルを開き、以下のコマンドを実行する：
    ```bash
    npx localtunnel --port 5173
    ```
3. ターミナルに表示される `your url is: https://xxxx.loca.lt` にスマホからアクセスする。
    - ※ プレビューを終了する際は、ターミナルで `Ctrl + C`

ビルド：

```bash
pnpm build
```
